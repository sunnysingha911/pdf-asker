# 📄 PDF QA Engine

Turn static PDFs into intelligent, searchable knowledge.  
Ask questions and get accurate answers powered by AI.

---

## 🚀 Overview

This project is a **Retrieval-Augmented Generation (RAG)** based system that allows users to:

- Upload PDF documents  
- Convert them into embeddings  
- Store and search using FAISS  
- Ask natural language questions  
- Get context-aware answers using LLMs (via Ollama)

---

## 🧠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- FastAPI (Python)

### AI / ML
- Ollama (LLM inference)
- Sentence Transformers (embeddings + reranking)

### Database
- FAISS (vector similarity search)

### PDF Processing
- PyPDF
